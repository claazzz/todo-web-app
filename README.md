# 📋 Веб-менеджер задач (To-Do App)

Простое и функциональное веб-приложение для управления списком задач с сохранением в базе данных SQLite. Проект написан на Python с использованием микрофреймворка Flask.

## 🚀 Возможности
- ➕ Добавление новых задач с заголовком и описанием
- ✅ Отметка задач как выполненных
- 🗑️ Удаление задач
- 💾 Автоматическое сохранение всех задач в базу данных SQLite
- 🎨 Чистый и адаптивный веб-интерфейс

## 🛠️ Стек технологий
- **Python 3.8+**
- **Flask** — веб-фреймворк
- **SQLite** — встроенная база данных
- **HTML + CSS** — интерфейс
- **Jinja2** — шаблонизатор для динамических страниц
